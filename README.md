# Contacts

## Rodando o projeto

Clonar repositório:
```
git clone https://github.com/Caiocbeleza/Contacts
```

Entrar na pasta do Projeto:
```
cd Contacts
```


Para rodar o server:
```
cd contacts
cd server
```
```
npm start
```

Para rodar o projeto vue:
```
npm run serve
```

Acessar aplicação:
```
http://localhost:8080/
```
